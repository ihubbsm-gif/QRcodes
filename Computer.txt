<!-- PRINCIPLE -->
<div class="card purple">
<div class="card-header">
<i class="fas fa-lightbulb"></i>
Principle
</div>
<div class="card-body">
<ul>
<li>A computer works on the Input → Process → Output (IPO) cycle.</li>
<li>It accepts data through input devices, processes it using the Central Processing Unit (CPU) according to stored instructions, stores the processed information in memory, and displays the results through output devices.</li>
<li>All operations are performed using binary language (0s and 1s) with the help of electronic circuits.</li>
</ul>
</div>
</div>

<!-- WORKING -->
<div class="card orange">
<div class="card-header">
<i class="fas fa-cog"></i>
Working
</div>
<div class="card-body">
<ul>
<li>Data is entered into the computer through an input device.</li>
<li>The CPU processes the data according to the given instructions.</li>
<li>Memory stores the required data during processing.</li>
<li>The processed information is displayed through an output device.</li>
<li>The information can also be stored for future use.</li>
</ul>
